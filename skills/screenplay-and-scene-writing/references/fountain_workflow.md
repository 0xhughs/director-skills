# Fountain Workflow

Use Fountain when the user wants plain-text screenplay output that can be imported into screenwriting tools.

## Core syntax

```text
INT. LOCATION - NIGHT

Action lines describe visible and audible behavior in present tense.

CHARACTER
Dialogue appears under the character name.

(brief parenthetical only when needed)
Dialogue continues.
```

## Rules for AI-assisted drafting

- Write one scene or beat at a time to preserve continuity and voice.
- Keep action lines filmable; do not include internal thoughts unless represented by action, sound, or image.
- Use parentheticals sparingly. They guide performance; they are not prose commentary.
- Keep camera directions out of the screenplay unless the user explicitly requests a shooting script.
- Put visual anchors, prompt notes, and shot ideas in a separate handoff block after the Fountain scene.

## Page/runtime note

Standard screenplay pages roughly map to screen minutes, but AI shorts should still be planned by beats and shot duration because generated clips have model-specific time limits.
