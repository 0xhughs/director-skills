# Example: Image Prompt to Video Prompt

Use the uploaded keyframe as the exact visual anchor. Preserve the brass key, the guard's hand position, chipped green paint, and cool fluorescent palette. Animate the key trembling once, then turning slowly in the lock. Camera remains macro and locked-off. The metal clicks, cloth sleeve shifts softly, fluorescent hum continues. End with the key fully turned. No extra fingers, no changing key shape, no background morphing.
