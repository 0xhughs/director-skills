# Visual Bible System

A visual bible turns creative direction into reusable prompt anchors.

## Sections
- Project style premise.
- Palette rules: dominant, accent, forbidden colors.
- Camera rules: preferred shot scale, movement, lens distance.
- Lighting rules: source, contrast, color temperature.
- Texture rules: skin, fabric, surfaces, atmosphere.
- Design rules: props, architecture, costume language.
- Model notes: reference images, seeds, aspect ratios, known failure risks.

## Prompt packet
For each shot, include only the anchors that matter. Avoid pasting the entire bible into every prompt.
