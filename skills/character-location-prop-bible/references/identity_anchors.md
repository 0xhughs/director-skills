# Identity Anchors

## Character anchor fields
- Name/ID.
- Age range and presence.
- Face shape, hair, eyes, skin details.
- Silhouette and posture.
- Wardrobe baseline and materials.
- Distinctive prop or gesture.
- Emotional baseline.
- Forbidden drift: changes that would break identity.

## Reference sheet guidance
Generate neutral, well-lit, front/three-quarter/profile views before dramatic variants. Avoid extreme expressions, heavy shadows, or occluding props in the reference sheet.
