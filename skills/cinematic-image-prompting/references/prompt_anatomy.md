# Prompt Anatomy

Use the three-layer model.

## Layer 1: Creative intent
Story purpose, character emotion, theme pressure, audience experience, genre, and scene function.

## Layer 2: Cinematic execution
Shot type, framing, camera angle, lens, lighting, blocking, production design, texture, realism level, audio if video, and continuity anchors.

## Layer 3: Model-specific adaptation
Prompt syntax, length, parameters, negative support, reference handling, aspect ratio, duration, resolution, seed/reference controls, known failure modes.

## Front-loading rule
Place the identity, action, setting, and desired medium in the opening sentence. Put secondary style and constraints later.

## Prompt economy
Remove filler such as "we see" or repeated adjectives. Keep every clause tied to a visible or audible result.
