# Image Prompt Components

## Universal image stack
1. Purpose: character sheet, scene still, keyframe, poster, prop, location, product, concept art.
2. Subject: stable identity, silhouette, age range, wardrobe/materials, pose or action.
3. Context: location, time, weather, story moment, production design.
4. Composition: shot scale, angle, framing, aspect ratio, negative space.
5. Camera/lens: focal length feel, depth of field, sensor/format only if useful.
6. Lighting: source, direction, quality, contrast, color temperature.
7. Palette/style: genre, color rule, medium, realism level.
8. Texture: skin, fabric, surface, atmospheric details.
9. Continuity: bible IDs, prop state, wardrobe, location state.
10. Constraints: model-appropriate negatives or positive restatements.

## Prompt formats
- Natural prose: best for LLM-mediated models.
- Priority stack: best when front-loading and clarity matter.
- Shot card: concise, useful for production packages.
- Model-specific: route through model-adaptation.

## Completeness checklist
A prompt is complete when it answers: what is seen, why it matters, where it is, how it is framed, how it is lit, what texture makes it real, and what must stay consistent.
