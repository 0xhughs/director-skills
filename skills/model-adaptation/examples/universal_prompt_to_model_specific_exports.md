# Example: Universal Prompt to Model-Specific Exports

## Universal brief
Night guard turns a brass key in a basement storage door. Mood: dread and grief. Macro close-up, cool fluorescent light, realistic texture, 8-second I2V.

## Grok/Aurora export
Prompt: Preserve the uploaded macro keyframe. The worn brass key stamped 12B trembles once between the guard's fingers, then turns slowly in the lock. Locked-off macro shot, cool green fluorescent practical light, realistic metal scratches and skin texture, quiet suspense. Audio: soft sleeve movement, metallic click, fluorescent hum. Stable hand shape and unchanged key design throughout.
Settings: I2V, 8s, 16:9 if supported.
Unknowns: current negative field support.

## Kling export
Prompt: Use source image as start frame. Macro locked-off shot of the brass key turning slowly in the lock; preserve hand, key, door paint, and lighting. Cool fluorescent basement mood, realistic metal texture. Audio added in post if current tier lacks native audio.
Negative prompt: extra fingers, changing key shape, background morphing, blurry lock.
Settings: I2V, 5-10s depending tier, 16:9.
