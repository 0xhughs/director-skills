# Common Failure Modes

| Symptom | Likely cause | First fix |
|---|---|---|
| Identity drift | no reference or weak anchors | use character sheet/reference workflow |
| Waxy plastic look | missing texture/light physics | add skin/fabric/surface detail and physical shadows |
| Camera chaos | too many moves | keep one dominant move |
| Subject stops moving | action not continuous | specify continuous action for full duration |
| Bad hands | close hand task | reframe wider, simplify interaction |
| Text gibberish | model text limitation | choose text-capable model or simplify text |
| Prompt ignored | important detail buried | front-load key instruction |
| Over-stylized output | too many style anchors | choose one dominant style plus execution details |
| Rejection | unsafe/IP/policy-sensitive content | remove or generalize sensitive content, do not bypass |
