# Revision Loops

## Minimal loop
1. Define intended result.
2. Observe actual result.
3. Name the single failure that matters most.
4. Change one variable.
5. Predict the effect.
6. Test and record.

## Structured A/B loop
- A: preserve prompt, adjust camera/lens.
- B: preserve prompt, adjust light/palette.
- C: preserve prompt, adjust reference/continuity.
- D: preserve prompt, adjust model/settings.

Use A/B only when the original output is close. For broken prompts, rebuild the universal brief first.
