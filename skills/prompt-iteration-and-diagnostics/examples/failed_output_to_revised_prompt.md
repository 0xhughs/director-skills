# Example: Failed Output to Revised Prompt

## Failure
The video changed the key into a silver modern key and added extra fingers.

## Diagnosis
Reference preservation and hand anatomy failed. The prompt over-described mood but under-specified immutable prop and hand constraints.

## Revision
Use the uploaded image as the exact visual anchor. Preserve the worn brass key stamped 12B, same shape, same tarnished color, same guard hand with five fingers, same lock and chipped green door paint. Animate only a small tremble followed by one slow clockwise turn. Locked-off macro camera. Cool fluorescent light. Constraints: no key redesign, no extra fingers, no background morphing.
