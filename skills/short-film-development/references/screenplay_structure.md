# Screenplay Structure for Shorts

## Micro three-act pattern
- Setup: introduce person, pressure, place, and visual rule.
- Turn: the character chooses or discovers something that changes the pressure.
- Payoff: consequence lands in an image, action, or reversal.

## Beat types
- Want beat: character pursues a concrete goal.
- Resistance beat: another person, system, environment, or self blocks them.
- Discovery beat: new information changes the strategy.
- Cost beat: a choice creates visible consequence.
- Image beat: a visual detail carries meaning without explanation.

## Compression rules
- Start late, leave early.
- Combine exposition with conflict.
- Make props and locations do story work.
- Avoid backstory unless it changes present action.
